AI-Powered Personal Finance Advisor
Overview
AI-Powered Personal Finance Advisor is a Full-Stack web application that helps users analyze their
financial behavior and optimize their savings using Artificial Intelligence and Machine Learning.
Features
• User Authentication (JWT)
• Financial Savings Prediction (Machine Learning Model)
• Expense Analysis by Category
• AI Financial Chatbot Integration
• Modern Responsive UI (Next.js + Tailwind)
Tech Stack
• Frontend: Next.js, React, TypeScript, Tailwind CSS
• Backend: Django, Django REST Framework
• Machine Learning: Python, Scikit-learn
• Database: PostgreSQL
Installation (Without Docker)
1. Clone the repository:
git clone https://github.com/BOULATARM/AI-Powered-Personal-Finance-Advisor.git
2. Backend setup:
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
3. Frontend setup:
cd frontend
npm install
npm run dev
